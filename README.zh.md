# 视频代码索引仓库

本仓库是我在 TikTok / Instagram / YouTube 发布的  
**嵌入式与 IoT 教程视频的统一代码入口**。

为了避免不同技术栈互相干扰，代码按 **生态系统** 分开维护，  
你只需要下载你真正用得到的部分。

---

## 🚀 代码生态

### 🔹 Arduino
基于 Arduino IDE，适合入门与教学。

仓库地址：  
https://github.com/yirenova/video-arduino-labs

---

### 🔹 ESP-IDF
基于 ESP-IDF 的专业 ESP32 开发示例。

仓库地址：  
https://github.com/yirenova/video-esp-idf-labs

---

### 🔹 App Inventor
用于 IoT 控制与仪表盘的低代码 App 示例。

仓库地址：  
https://github.com/yirenova/video-inventor-labs

---

## 📚 使用方式

1. 找到你感兴趣的视频
2. 在索引表中查看对应 **用例编号**
3. 进入对应生态的仓库
4. 按用例目录中的 README 操作即可复现
